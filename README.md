## Descrição

O sistema será um site com Python como backend, sendo um site de aluguel de salas e computadores de Lan Houses Gamers, para que os clientes possam jogar e organizar campeonatos.

---

## Tecnologias

- Banco de Dados
- CRUD
- LGPD para dados sensíveis
- Python como Backend
- Tailwind e WEB para Frontend
- Acesso a APIs de Geolocalização
